# Tkinter
  The tkinter package ("Tk interface") is the standard Python interface to the Tcl/Tk GUI toolkit.
  To check the Tk version, create a file called tk_version.py:
  
  ```bash
  # tk_version.py
  import tkinter
  print(tkinter.TkVersion)
  $ python3 tk_version.py 
  9.0

# Abbreviation:

  * Tk - Toolkit
    
  * inter - Interface
